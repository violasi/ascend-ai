# SWE Job Prep Dashboard

AI-powered interview preparation for software engineers targeting AI startups and FAANG++ companies.

## Features

- **Real job listings** fetched from Greenhouse, Lever, and Ashby ATS APIs
- **5-type prep plans** generated by Claude Opus per job:
  - 📝 Coding Plan — DSA patterns, 6-week study plan, must-do LeetCode problems
  - 🏗 System Design — company-specific questions, level-calibrated depth
  - 🎯 Behavioral — STAR stories, LP mapping, bar raiser playbook
  - 🏢 Company Tips — loop structure, TC by level, cultural signals
  - ⚡ Edge Tech — top-of-band skills, sprint plan, promo criteria
- **31 companies** — 11 FAANG++ / 10 AI Startups
- **Level detection** (L3–L7+) from job titles
- **Auto-refresh** every 6 hours

## Quick Start

```bash
cp .env.example .env
# Edit .env with your ANTHROPIC_API_KEY
docker compose up -d
```

- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API docs: http://localhost:8000/docs

## Architecture

```
Frontend (Next.js 14 + Tailwind)
    ↓
Backend (FastAPI + asyncpg)
    ↓
PostgreSQL (jobs cache + prep plans)
    ↓
Greenhouse / Lever / Ashby APIs  (job data)
Anthropic Claude Opus             (prep generation)
```
